🏋️ Cross-training guide

🔄 Cross trainer — Mon & Fri

  • Easy aerobic: 20–30 min, RPE 4/10 (could sing)
  • Why: Same cardio as running, zero impact on joints
  • Swap rule: If you miss a run day, 45 min cross trainer replaces it
  • Progression: Add 5 min every 2 weeks as you build fitness

🪢 Resistance bands — before every run

  • Clamshells: 3 × 15 each side
  • Lateral walks: 3 × 20 steps each way
  • Monster walks: 3 × 15 steps fwd/back
  • Hip circle squat: 2 × 10
5–10 min pre-run. Activates glutes — #1 injury prevention for new runners.

🔔 Kettlebell — Wed & Sat (Strength A)

  • Start with: 12–16 kg
  • KB swings: 3 × 15 (posterior chain king)
  • Goblet squat: 3 × 10
  • KB deadlift: 3 × 10
  • Single-leg RDL: 3 × 8 each leg
Rest 60–90 s between sets. Done in ~25 min.

🏋️ Dumbbells — Sat (Strength B)

  • Start with: 8–12 kg pair
  • Reverse lunge: 3 × 10 each leg
  • Step-ups: 3 × 10 each leg (use a chair)
  • Hip thrust: 3 × 12 (bodyweight ok to start)
  • Bent-over row: 3 × 12 (for posture)
Alternate A/B each Saturday to hit all muscle groups weekly.
📌 Which to buy first? If you only buy one thing: a 16 kg kettlebell — KB swings alone will transform your running strength. Bands are second (cheap and essential for warm-up). Dumbbells third if you want the full B session.

⚠️ Load rule: Strength sessions reduce to 1×/week in the 10k Prep and Half Build phases, and drop to mobility-only during Taper. The schedule above updates automatically each week.
